Разработать программу для работы с текстовыми файлами.
Основные действия пользователя:
∙ Просмотр дерева файлов рабочей директории;
∙ Добавление/редактирование/удаление файла;
∙ Фильтрация файлов по названию (включая использование glob-шаблонов);
∙ Фильтрация файлов по содержимому;
∙ Просмотр статистики.
Требования:
∙ Автоматическое обновление представления дерева файлов в интерфейсе при добавлени-
и/удалении файлов в рабочей директории проекта (включая ситуации, когда происходит
добавление/удаление файла не через функционал программы);
∙ Фильтрация файлов по содержимому предполагает точный поиск строки в файле;
∙ Статистика должна включать в себя число слов по каждому файлу рабочей директории
и суммарное число слов в рабочей директории;
∙ Под редактированием подразумевается простое изменение символов текста, без каких-
либо возможностей по форматированию, стилизации, выравниванию и пр.;
∙ Файлы — простые текстовые файлы с заданным расширением и кодировкой;
∙ Путь до рабочей директории, расширение и кодировка файлов должны задаваться в кон-
фигурационном файле программы.
